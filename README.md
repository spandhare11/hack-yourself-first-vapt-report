# Web Application Penetration Test – Hack Yourself First

## Overview

This repository contains a professional Web Application Penetration Test performed against the intentionally vulnerable **Hack Yourself First** application.

The assessment was conducted following:

- OWASP Web Security Testing Guide (WSTG)
- OWASP Top 10 (2021)
- PTES Methodology
